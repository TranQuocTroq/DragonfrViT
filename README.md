## Overview

DragonFruitViT implements a **Vision Transformer (ViT-B/16) entirely from scratch** using only PyTorch base modules — no pretrained weights, no external ViT libraries. The model classifies five dragon fruit disease categories from field images collected by the team.

The project also includes a systematic **ablation study** that isolates the contribution of each key ViT component (positional embedding, class token, MLP block) by training and evaluating six architectural variants.

---

## Task

Given a field image of a dragon fruit plant, classify it into one of five categories:

| Label | Disease |
|---|---|
| DomNau | Brown spot |
| KhoeManh | Healthy plant |
| NamCanh | Stem canker |
| ThanThu | Anthracnose |
| ThoiBe | Base rot |

**Dataset:** G-Dragon — self-collected field images, split 80 / 10 / 10 (train / val / test).

---

## Method

### ViT-B/16 Architecture (from scratch)

All components are implemented using ``nn.Module`` with no external dependencies:

```
Input image [B, 3, 224, 224]
    → Patch Embedding   (Conv2d, patch=16×16 → 196 tokens × 768 dim)
    → Class Token       (prepended learnable token)
    → Positional Embedding (learnable, 197 × 768)
    → 12× Transformer Block
        ├── LayerNorm
        ├── Multi-Head Self-Attention (12 heads, scale = head_dim^-0.5)
        ├── LayerNorm
        └── MLP (768 → 3072 → 768, GELU activation)
    → LayerNorm
    → Class Token output → Linear head → [B, num_classes]
```

### Training

- Optimizer: AdamW (lr=1e-3, weight_decay=1e-4)
- Scheduler: ReduceLROnPlateau (factor=0.1, patience=2)
- Early stopping: patience=5 epochs
- Augmentation: RandomRotation ±15°, RandomCrop 224×224, RandomHorizontalFlip
- Normalization: ImageNet mean/std

## Results


### Ablation study

| Scenario | ACC | F1 |
|---|---|---|
| S1: Full ViT (baseline) | 0.8321 | 0.8384 |
| S2: No positional embedding | 0.8473 | 0.8443 |
| S3: No class token (GAP) | 0.8550 | 0.8571 |
| S4: No MLP block | 0.8702 | 0.8718 |
| S5: No pos embed + no class token | 0.8550 | 0.8606 |
| S6: No pos embed + no cls + no MLP | 0.8955 | 0.8914 |

## Usage

**Step 1 — Edit `config.yaml`**

Set `data.raw_dir` to the path of your dataset directory.

**Step 2 — Split data (run once)**
```bash
python preprocess.py --config config.yaml
```

**Step 3 — Train**
```bash
python train.py --config config.yaml
```

**Step 4 — Predict on a single image**
```bash
python predict.py --image path/to/image.jpg --config config.yaml
```

**Step 5 — Run ablation study**
```bash
python ablation_study.py --config config.yaml --epochs 20
```

**Step 6 — Visualize results**
```bash
python visualize.py --config config.yaml
```
